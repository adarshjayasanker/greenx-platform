# Greenx Pest Control Platform — Full-Stack MERN Application

## Overview

Greenx is a real-world business platform developed for a pest control service company based in Kochi.
The system was rebuilt from a static template into a modern full-stack application to improve user experience, automate business workflows, and enable lead management.

This platform is actively used to capture customer inquiries and manage business operations.

---

## Key Features

* **Modern Frontend UI** built using React and Tailwind CSS
* **Admin Panel** to manage services, testimonials, and website content
* **Lead Generation System** to capture and store customer inquiries
* **Testimonial Approval Workflow** for content moderation
* **WhatsApp Integration** for direct customer communication
* **Admin Authentication System** for secure access

---

## Tech Stack

* **Frontend:** React, Tailwind CSS
* **Backend:** Node.js, Express.js
* **Database:** MongoDB

---

## System Architecture

The application follows a typical client-server architecture:

* React frontend communicates with backend APIs
* Express server handles business logic and routing
* MongoDB stores application data such as leads, services, and testimonials
* Admin panel interacts with protected routes secured via authentication

---

## Key Engineering Decisions

* Migrated from EJS-based rendering to a React-based frontend for better scalability and UI control
* Designed RESTful APIs for modular and maintainable backend structure
* Implemented database-driven content management to eliminate hardcoded updates
* Structured admin workflows to separate public and internal system behavior
